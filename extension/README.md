# RSVP Reader - Edge 扩展安装指南

## 安装步骤

### 方法一：开发者模式安装（推荐）

1. **打开 Edge 浏览器**
   - 启动 Microsoft Edge

2. **进入扩展管理页面**
   - 在地址栏输入：`edge://extensions/`
   - 或者点击右上角三个点 → 扩展

3. **启用开发者模式**
   - 在扩展页面左下角，打开"开发人员模式"开关

4. **加载扩展**
   - 点击"加载解压缩的扩展"
   - 选择 `extension` 文件夹（包含 manifest.json 的文件夹）
   - 点击"选择文件夹"

5. **使用扩展**
   - 安装成功后，在 Edge 工具栏会出现扩展图标
   - 点击图标即可打开 RSVP Reader 弹出窗口

### 方法二：打包安装

1. **打包扩展**
   - 在 `edge://extensions/` 页面，点击"打包扩展"
   - 选择 `extension` 文件夹
   - 生成 `.crx` 文件

2. **安装打包文件**
   - 将生成的 `.crx` 文件拖拽到 Edge 扩展页面即可安装

## 图标说明

如果扩展显示默认图标，你可以：

1. **创建图标文件**（可选）：
   - 创建以下尺寸的 PNG 图标：
     - `icon16.png` (16x16 像素)
     - `icon32.png` (32x32 像素)
     - `icon48.png` (48x48 像素)
     - `icon128.png` (128x128 像素)
   - 将图标文件放在 `extension` 文件夹中

2. **或者使用在线工具生成图标**：
   - 访问 https://www.favicon-generator.org/ 等在线工具
   - 上传一个图标，生成多尺寸版本

## 使用说明

1. 点击 Edge 工具栏中的 RSVP Reader 图标
2. 在弹出窗口中：
   - 输入文本或上传 `.txt` 文件
   - 调整阅读速度（100-900 WPM）
   - 点击"Start"开始阅读
   - 使用"Pause"暂停，"Replay"重新播放

## 功能特性

- ✅ 快速阅读（RSVP 技术）
- ✅ ORP（最优识别点）高亮显示
- ✅ 可调节阅读速度
- ✅ 支持文本输入和文件上传
- ✅ 实时状态显示

## 故障排除

如果点击图标没有弹出窗口：

1. 检查扩展是否已启用（在 `edge://extensions/` 页面）
2. 检查是否有错误提示（点击扩展的"详细信息"查看）
3. 尝试重新加载扩展（点击扩展卡片上的刷新图标）
4. 确保所有文件都在 `extension` 文件夹中：
   - manifest.json
   - popup.html
   - popup.js
   - renderer.js
   - style.css
